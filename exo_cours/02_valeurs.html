Valeurs

Observez le programme valeurs.js et tentez de prévoir les valeurs finales de chaque variable.

var a = 2; Cette variable indique la valeur de a

a = a - 1; Cette valeur est égal à elle-même soustrait à 1, donc a est égal à 1

a++; (regardez incrémentation sur internet); une incrémentation permet d'augmenter la valeur de 1, donc a est égal à 2

var b = 8; cette variable indique la valeur de b

b += 2; on ajoute 2 à la variable b, donc b est égal à 10

var c = a + b * b; la variable c est égale à l'opération (a + b * b), la variable c est égale à 102

var d = a * b + b; la variable d est égale à l'opération (a * b + b), la variable d est égale à 30

var e = a * (b + b); la variable e est égale à l'opération (a * (b + b)), la variable e est égale à 40

var f = a * b / a; la variable f est égale à l'opération (a * b / a), la variable f est égale à 10

var g = b / a * a; la variable g est égale à l'opération (b / a * a), la variable g est égale à 10

Vérifiez vos prévisions en créant ce programme puis en le déboguant pour observer les valeurs des variables.
